# DevTask AI Backend

Backend API for DevTask AI - A unified developer productivity platform with AI-powered bug fix suggestions.

## Tech Stack

- Node.js + Express
- MongoDB Atlas
- Gemini AI
- AWS S3
- JWT Authentication

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file from `.env.example` and fill in your credentials

3. Start development server:
```bash
npm run dev
```

4. Start production server:
```bash
npm start
```

## API Documentation

Base URL: `/api/v1`

See design document for complete API endpoint documentation.
